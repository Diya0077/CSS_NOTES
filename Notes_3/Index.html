  • Now we will learn transforation and animation
    • Transformation is used to rotate, move, skew or scale elements. It used to apply a 2D or 3D transformation to an element.
  • CSS 2D transform methods:
    • You can use the following 2-D transforms in CSS:

    1. rotate
    2. scale
    3. matrix

  • CSS 3-D transform methods:

    1. rotate x
    2. rotate y
    3. rotate z

  • CSS Animation

    • It used to animate CSS properties with more control. We can use @keyterms rule to change the animation from a given style to a new style.All the animation properties from 1 to 6 can be applied. We can use % values to indicate what should happen when a certain percent of animation is completed.

  • Properties to Add Animations

    1. Animation name: Name of the animation
    2. Animation duration: How long does the animation run
    3. Animation timing function: Determines speed curve of the animation
    4. Animation delay: Delay for the start of an animation
    5. Animation direction: Specifies the direction of the animation
