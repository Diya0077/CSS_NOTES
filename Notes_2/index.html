    • Now We will learn about using fronts and display

  • The display property
    • The CSS display property is used to determine whether an element is treated as a block/inline.

  • Display:inline
    • Takes only the space required by the element. No line breaks before and after. detting width,height,mapping and padding not allowed.
  • Display:Block
    • Takes full space available in width and leaves a newline before and after the element.
  • Display:Inline-block
    • Similar to inline but setting height, width, margin and paddind is allowed.
  • Text Align property
    • It used to set the horizontal alignment of a text.
  • Text Decration property
    • We use it to decorate the text. it can be overline, line-through, underline and none.
  • Text transform property
    • It used to specify uppercase and lowercase letterrs in a text.
  • Line height property
    • It used to specify the space between lines.

  • Now I will say you about size, position and list
    • There are some units for describing size like px, rem, cm, vw, percentage etc.
  • What is called pixel?
    • Pixel is a unit which is relative to the viewing device
  • Relative lengths
    • These units are relative to the other length property. Folloeing are some of the most commonly used relative lengths.
    1.em - unit relative to the parent font size.
    2.rem - unit relative to the root font size.
    3.vw - unit relaative to 1% viewport width.
    4.vh - unit relaative to 1% viewport height.
    5.% - unit relative to the parent element.
  • Min/Max-Height/width Property
    • CSS has a min-height, max-height, min-width and max-width property. If the content is smaller than the minimum height will be applied.
  • The Position Property
    • Used to manupulate the locaton of an element, following are the possible values:
    Static: the default position. top/bottom/left/right/ z-index has no effect.
    Relative: the default position. top/bottom/left/right/ z-index will work now. Otherwise the elemnet is in the flow of document like static.
    absolute: The element is removed from the flow and it is relatively positioned to its first non-static ancestor.
    Fixed: Just like absolute except the element is positoned relative to the broswer window
    Sticky: The element is positioned based on user's scroll position.
  • List-Style Property
    • The list style property is a shorthand for setting type,position and image.
  • z-index Property:
    • This property specifies the stock order of an element. It defines which layer will be above in case of overlapping elements.
